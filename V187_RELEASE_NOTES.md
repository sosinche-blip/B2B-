# V187 쿠폰 24시간 자동운영 안전본

## 추가

- 여러 쿠팡 쿠폰 선택형 24시간 자동운영
- 23:45·23:47·23:49 사전검증
- 23:50 쿠폰별 취소, 23:51 취소 성공 쿠폰만 생성·적용
- 다음 날 23:50 종료시간 지정
- 기존 계약·상품·옵션·할인조건·와우 여부 복사
- 쿠폰명 `YYYY-MM-DD` 날짜 자동 교체
- 쿠폰별 독립 실패 처리
- 즉시·10초·30초/30분 재시도
- Supabase 영구 재시도 및 실패 알림
- 수동 재실행, 확인 완료, 자동운영 중지
- 쿠팡 비동기 요청상태 폴링 확인
- 허용시간 초과 작업 실패 알림

## 보완

- 사전점검 표본 제한은 실패가 아닌 안내로 처리
- 자동운영 중지 시 대기·실행 중 재시도 취소
- 중지된 쿠폰의 만기 재시도 자동 차단
- Ncloud 파일 처리 코드를 제거하고 고정 IP 실행 역할만 유지

## 토스

토스쇼핑 공개 API 문서에서 쿠폰·프로모션 생성/취소 기능이 확인되지 않아 동일 자동화는 비활성 상태로 유지합니다. 임의 엔드포인트를 만들거나 브라우저 자동조작으로 대체하지 않습니다.
