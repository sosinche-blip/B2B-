# V191 변경사항

1. 선택 주문 목록에 구매수량을 추가했습니다.
2. 선택 주문 수집 성공 시 업체별 발주양식을 자동 분류하고 ZIP을 자동 다운로드합니다.
3. 신규 쿠폰 상단의 contractId·상품명·쿠폰명·할인구분·할인값·와우회원 입력박스를 제거했습니다.
4. 계약ID는 계약서 선택값을 사용합니다.
5. 옵션 조회표의 상품명/옵션명은 매핑자료 업체상품명으로 표시합니다.
6. 기존 상태·재고·확인결과 열을 상품명 입력·할인값·할인구분 입력으로 변경했습니다.
7. 쿠폰명은 입력한 상품명을 기준으로 날짜를 자동 부여합니다.
8. 자동운영 활성화 시 신규 쿠폰을 즉시 생성·적용합니다.
9. 즉시 생성 쿠폰은 같은 날 스케줄에서 다시 취소하지 않고 다음 날부터 정기 반복합니다.
10. 기존 활성·대기 쿠폰 선택취소와 V190 안전기능은 유지합니다.
