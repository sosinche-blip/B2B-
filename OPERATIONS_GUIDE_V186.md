# V186 운영 적용 순서

## 1. 주문수집

1. 간편운영에서 쿠팡 수집, 토스 수집 또는 쿠팡+토스 수집을 실행합니다.
2. 쿠팡 결제완료는 `ACCEPT`, 토스 결제완료는 `PAID` 상태만 수량에 포함합니다.
3. 상품준비중 수량은 쿠팡 `INSTRUCT`, 토스 `PREPARING_PRODUCT`를 별도로 표시합니다.
4. 주문관리 진단은 날짜별 상세 행을 조회기간·상태별 합계로 요약합니다.

## 2. 업체송장과 운송장 등록

1. 업체송장 선택을 눌러 택배사와 운송장번호가 입력된 여러 엑셀을 선택합니다.
2. 파일은 브라우저 앱 메모리에만 임시 보관합니다.
3. 쿠팡+토스 업로드를 눌러 상품준비중 주문과 매칭 미리보기를 확인합니다.
4. 정상 매칭 건만 최종 업로드합니다.
5. 완료 후 `쿠팡_토스_전체처리결과_YYYY-MM-DD.xlsx` 파일 1개가 다운로드됩니다.
6. 파일 안에는 처리요약, 전체처리결과, 쿠팡업로드, 토스업로드, 미매칭확인 시트가 있습니다.
7. 정상 완료 후 앱의 임시 송장파일은 삭제됩니다.

## 3. 배포 범위

- GitHub와 Cloudflare Pages·Worker에 V186 코드를 배포합니다.
- Ncloud에는 V186 전체 압축파일을 올리지 않습니다.
- 현재 실행 중인 Ncloud 고정 IP 게이트웨이는 그대로 유지합니다.
