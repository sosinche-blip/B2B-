# V184 업체송장 자동업로드·ZIP

## 변경사항

1. `업체송장 선택` 버튼으로 여러 업체 송장 파일을 R2 발주폴더에 저장합니다.
2. 선택한 업체 송장 파일명을 현재 작업 세션에 보존합니다.
3. `쿠팡+토스 업로드`가 상품준비중 주문을 조회하고 업체 송장과 자동 매칭합니다.
4. 쿠팡·토스 채널별 송장 API 업로드가 성공한 뒤 관련 파일 ZIP을 자동 다운로드합니다.
5. R2 ZIP API에 파일명 선택 기능을 추가해 이번 작업 관련 파일만 압축할 수 있습니다.
6. 입력파일 기반 처리와 API 상품준비중 재수집 처리 모두 확인표와 ZIP을 생성합니다.
7. Ncloud는 고정 IP API 게이트웨이 역할만 유지합니다.
