# V186 주문요약·단일 결과파일

## 변경사항

1. 간편운영의 쿠팡·토스 주문 수량을 결제완료 상태로 제한했습니다.
2. 간편운영에서 발주업체와 송장등록 준비 지표를 삭제했습니다.
3. 날짜별 쿠팡 진단 행을 조회기간, 상태, 합계, 주문 발생일 중심으로 요약합니다.
4. 반복되는 인증·요청·HTTP 응답 행은 API 호출 확인 1행으로 축약합니다.
5. 최종 송장 업로드 결과를 개별 엑셀 4개와 ZIP이 아닌 단일 엑셀로 변경했습니다.
6. 결과 파일명에 처리 날짜를 포함합니다.
7. 단일 엑셀에 처리요약, 전체처리결과, 쿠팡업로드, 토스업로드, 미매칭확인 시트를 포함합니다.
8. 기존 업체송장 브라우저 임시저장, 매칭 미리보기, 최종확인 흐름은 유지합니다.
