# V192 운영 안내

## 1. 선택 주문 수집

1. 쿠팡 또는 토스 결제완료 주문을 조회합니다.
2. 필요한 상품을 체크합니다.
3. `선택 주문 수집`을 누릅니다.
4. 옵션ID 매핑에 따라 업체별 발주양식과 채널 입력파일을 생성합니다.
5. 생성 결과는 ZIP으로 다운로드합니다.

## 2. 신규 쿠폰 등록

1. 계약서 목록에서 계약을 선택합니다.
2. API 옵션ID를 입력하고 조회합니다.
3. 각 옵션 행에서 상품명, 쿠폰명, 할인값, 할인구분을 입력합니다.
4. 옵션별 조건은 서로 달라도 됩니다.
5. `신규 쿠폰 사전검증`을 실행합니다.
6. 상단 `자동운영 활성화`를 누르면 옵션별로 독립 쿠폰을 즉시 생성·적용합니다.
7. 첫 정기 교체는 다음 날 23:50 취소, 23:51 재발행부터 시작합니다.

## 3. 실패 처리

- 옵션별 생성은 독립 처리합니다.
- 일부 옵션이 실패하면 성공한 쿠폰은 자동운영에 등록하고 실패 옵션은 입력목록에 남깁니다.
- 실패 사유는 화면에 표시하며 성공하지 않은 옵션을 수정해 다시 실행할 수 있습니다.
- 활성·대기 쿠폰 취소는 목록 조회 후 체크한 쿠폰만 실행합니다.

## 4. 저장 구조

- 설정·자동운영·재시도·실패이력: Supabase
- 웹과 API 라우팅: Cloudflare Pages·Worker
- 파일: 브라우저 다운로드 또는 Cloudflare R2
- 쿠팡·토스 고정 IP 호출: Ncloud 최소 게이트웨이
