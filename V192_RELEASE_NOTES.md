# V192 변경사항

1. 신규 쿠폰 행에 `쿠폰명 입력` 추가
2. 상품명·쿠폰명·할인값·할인구분을 옵션별 독립 입력
3. 동일값 강제검증 제거
4. 옵션마다 독립 couponId 생성 및 반복템플릿 저장
5. 일부 실패 시 성공 옵션만 저장하고 실패 옵션은 화면에 유지
6. 중복 쿠폰양식·판매가동기화·수동 쿠폰편집 UI 제거
7. 스케줄러의 위험한 수동 쿠폰 생성·취소 중복 UI 제거
8. 사용하지 않는 함수와 과거 Windows/Ncloud 로컬 실행파일 제거
9. Cloudflare 앱과 Ncloud 게이트웨이 패키지 분리
