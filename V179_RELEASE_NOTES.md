# V179_RELEASE_NOTES

## 핵심 변경
- V178의 직접 Ncloud HTTP fallback 관련 코드를 제거했습니다.
- 별도 `API 502 점검` 패널을 제거하고 실행경로·환경변수 점검으로 단순화했습니다.
- 과거 버전 문서 중복본과 Windows CMD 보조파일을 배포 ZIP 루트에서 제거했습니다.
- 매핑 업로드 표준은 `채널/옵션ID/업체명/코드번호/업체상품명/원가/기본수량` 7개 열로 유지했습니다.
- `xlsx` 내장 파서는 유지하여 매핑 업로드 안정성을 보존했습니다.

## 유지 기능
- 주문 수집
- 옵션ID 자동 매핑
- 업체별 발주 ZIP 다운로드
- 업체 송장 업로드
- 송장 안전검증
- 쿠팡·토스 송장 업로드
- 쿠폰 자동화
- 서버 정리 및 운영 로그
