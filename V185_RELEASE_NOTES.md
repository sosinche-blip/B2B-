# V185 업체송장 임시매칭·최종확인

## 변경사항

1. `업체송장 선택`이 Cloudflare R2·Supabase·Ncloud에 파일을 저장하지 않도록 변경했습니다.
2. 선택한 여러 업체 송장과 파싱한 송장 행을 브라우저 앱 메모리에만 임시 보관합니다.
3. 새로고침 또는 브라우저 종료 시 임시 파일이 삭제됩니다.
4. 첫 번째 `쿠팡+토스 업로드`는 상품준비중 주문 조회와 매칭 미리보기만 수행합니다.
5. `최종 업로드`를 별도로 추가해 실제 쿠팡·토스 API 실행을 분리했습니다.
6. 정상 매칭 건만 업로드하고 미매칭 건은 업로드에서 제외합니다.
7. 쿠팡 결과, 토스 결과, 미매칭 확인, 전체 처리결과 엑셀을 생성합니다.
8. 결과 엑셀 4개를 묶은 ZIP을 함께 다운로드합니다.
9. API 실패 시 임시 송장과 미리보기를 유지해 재검토할 수 있습니다.
10. 성공 시 임시 송장을 자동 삭제하고 결과 재다운로드 버튼을 제공합니다.
