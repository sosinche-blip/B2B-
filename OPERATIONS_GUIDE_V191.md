# V191 운영 안내

## 1. 선택 주문 수집

1. 쿠팡 또는 토스 주문조회를 실행합니다.
2. 결제완료 목록에서 상품명·옵션명과 구매수량을 확인합니다.
3. 필요한 주문을 체크하고 `선택 주문 수집`을 누릅니다.
4. 옵션ID 매핑을 기준으로 업체별 발주양식을 자동 생성합니다.
5. 업체별 파일, 쿠팡·토스 상품준비중 입력파일, 매핑검증표가 포함된 ZIP 다운로드가 시작됩니다.
6. 미매핑이 있으면 발주파일 대신 검증파일을 확인합니다.

## 2. 신규 쿠폰 등록

1. 계약서 목록에서 `신규생성 선택`을 누릅니다.
2. 쿠팡 API 옵션ID를 입력하고 조회합니다.
3. `매칭자료 업체상품명`을 확인합니다. 값이 없으면 매핑관리에서 먼저 등록합니다.
4. 각 선택 옵션의 `상품명 입력`, `할인값`, `할인구분`을 입력합니다.
5. 정률 할인일 때만 최대 할인금액을 입력합니다.
6. `신규 쿠폰 사전검증`을 실행합니다.
7. 상단 `자동운영 활성화`를 누르면 신규 쿠폰이 즉시 생성·적용됩니다.
8. 첫 정기 교체는 다음 날 23:50 취소, 23:51 재발행부터 시작합니다.

한 번에 선택한 옵션은 동일 상품명·할인구분·할인값을 사용해야 합니다. 다른 조건은 나누어 등록합니다.
