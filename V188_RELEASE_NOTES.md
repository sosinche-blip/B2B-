# V188 API 현황·선택주문·모바일 바로가기 운영본

## 주요 변경사항

1. 앱 시작 시 쿠팡·토스 결제완료·상품준비중 수량 자동조회
2. 현황 4개를 모바일에서도 한 줄 표시
3. 현황 수동 새로고침 추가
4. 쿠팡·토스 수집 버튼을 결제완료 주문조회 방식으로 변경
5. 체크박스와 상품명/옵션명만 보여주는 선택수집 목록 추가
6. 체크한 주문만 매핑·발주 처리
7. 업체 바로가기에 로그인ID 필드 추가
8. 모바일 업체 바로가기 클릭 시 로그인ID 클립보드 복사
9. 비밀번호는 브라우저 비밀번호 관리자 사용
10. 쿠폰 선택·검증·활성화·중지·삭제 Supabase 자동저장
11. 쿠폰관리 수동 브라우저·서버 저장 버튼 제거

## 유지사항

- V187 쿠팡 24시간 쿠폰 자동운영
- V187 Ncloud 고정 IP 게이트웨이
- 업체송장 브라우저 임시저장·매칭·최종 업로드
- 토스 쿠폰 자동화 비활성화
