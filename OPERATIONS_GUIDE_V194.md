# V194 운영 안내

## 일일 시작

1. 간편운영에서 `운영점검 새로고침`을 누릅니다.
2. 결제완료와 상품준비중 현황을 확인합니다.
3. 미해결 실패가 있으면 실패 재처리 센터에서 해당 단계만 다시 실행합니다.

## 주문수집과 주소검사

1. 쿠팡 또는 토스 주문조회를 실행합니다.
2. 필요한 상품을 체크하고 선택 주문 수집을 실행합니다.
3. 주소 품질검사에서 `차단` 건이 없는지 확인합니다.
4. 차단 건은 주문 원문 또는 판매처 배송지를 확인한 뒤 다시 수집합니다.
5. `주의` 건은 동·호수·층이 원래 없는 주소인지 확인합니다.

## 실패 재처리

- 주문조회 실패: 같은 채널 결제완료 조회를 다시 실행합니다.
- 주문수집 실패: 해당 채널 주문수집과 발주 생성 단계를 다시 실행합니다.
- 발주 실패: 현재 검증된 발주대상으로 전체 발주파일 생성을 다시 실행합니다.
- 송장 미리보기 실패: 선택된 임시 업체송장을 유지한 상태로 매칭을 다시 실행합니다.
- 송장 업로드 실패: 기존 미리보기가 남아 있을 때 최종 업로드만 다시 실행합니다.
- 쿠폰 실패: Supabase에 저장된 실패 단계부터 재시도합니다.

재시도 조건이 사라졌거나 입력파일이 없는 경우 상태가 `수동확인`으로 바뀝니다.

## 일일 마감

1. 미매핑, 주소 차단, 미해결 실패가 0건인지 확인합니다.
2. `마감보고서 다운로드`를 눌러 엑셀을 보관합니다.
3. 필요 시 서버 1일 임시저장으로 현재 작업자료와 일반 실패목록을 보관합니다.
