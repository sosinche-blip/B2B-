# V194 운영 안내

## 결제완료 주문 수합

1. 주문관리에서 `결제완료`를 누릅니다.
2. 쿠팡 또는 토스 `주문조회`를 누릅니다.
3. 필요한 주문을 체크합니다.
4. `선택 주문 수집`을 누릅니다.
5. 업체별 발주양식과 채널 상품준비중 입력파일이 생성되고 ZIP 다운로드가 시작됩니다.
6. 정상 발주된 주문만 상품준비중 변경을 요청합니다.

## 상품준비중 주문 수합

1. 주문관리에서 `상품준비중`을 누릅니다.
2. 쿠팡 또는 토스 `주문조회`를 누릅니다.
3. 필요한 주문을 체크합니다.
4. `선택 주문 수집`을 누릅니다.
5. 선택 주문이 현재 주문목록과 합쳐지고, 업체송장 매칭·송장업로드 대상에 반영됩니다.

상품준비중 주문은 이미 발주 후 단계이므로 다음 작업은 실행하지 않습니다.

- 업체별 발주 ZIP 재생성
- 상품준비중 상태 변경 재요청

## 주문 상태 갱신

같은 주문이 기존 결제완료 행으로 저장돼 있어도 상품준비중으로 다시 조회하면 주문번호와 옵션ID 기준으로 기존 행을 최신 상태·주소로 갱신합니다. 다른 주문은 삭제하지 않습니다.

## 주소 확인

V193부터 다음 전체주소 보존 규칙을 유지합니다.

```text
경남 남해군 남해읍 화전로 10 (남해리)
+ 101동 1203호
= 경남 남해군 남해읍 화전로 10 (남해리) 101동 1203호
```
