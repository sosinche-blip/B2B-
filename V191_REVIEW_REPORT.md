# V191 3회 검토 결과

## 1차 기능 검토

- 선택 주문 상품명·옵션명·구매수량 표시
- 옵션ID 매핑 기반 업체별 발주양식 자동 분류
- 선택 주문 결과 ZIP 자동 다운로드
- 쿠폰 옵션행 입력 구조 반영

## 2차 안전 검토

- 매칭자료 업체상품명 누락 시 신규 쿠폰 사전검증 차단
- 옵션별 할인값·할인구분 검증
- 한 번에 서로 다른 쿠폰조건 혼합 등록 차단
- 신규 쿠폰 활성화 즉시 생성·적용
- 즉시 생성 당일 정기 취소 중복 방지

## 3차 기술 검토

- 웹 프로덕션 빌드 통과
- Worker TypeScript 검사 통과
- Ncloud build-only 검사 통과
- V191 전용 검증 통과
- Ncloud V187 최소 게이트웨이 역할 유지
