# V178 모바일 운영 점검표

## 매핑 업로드

- [ ] 매핑 양식 열이 `채널, 옵션ID, 업체명, 코드번호, 업체상품명, 원가, 기본수량` 순서인지 확인
- [ ] 채널 값이 쿠팡 또는 토스인지 확인
- [ ] 옵션ID가 숫자처럼 보여도 텍스트로 보존되는지 확인
- [ ] 매핑 업로드 후 적용 행 수가 표시되는지 확인
- [ ] 미매핑 건수가 줄었는지 확인

## API 502 점검

- [ ] `API 502 점검` 버튼 실행
- [ ] Worker 자체 응답이 정상인지 확인
- [ ] `NCLOUD_API_BASE`가 현재 유효한 Tunnel/도메인인지 확인
- [ ] 임시 trycloudflare 주소라면 바뀌지 않았는지 확인
- [ ] Ncloud 서버에서 8080 API가 살아 있는지 확인

## 쿠팡·토스 진단

- [ ] 쿠팡진단 실행
- [ ] 토스진단 실행
- [ ] IP확인 실행
- [ ] 502가 나오면 쿠팡·토스 API보다 Worker/Tunnel/Ncloud 경로를 먼저 점검
- [ ] 401/403이 나오면 키 또는 허용 IP를 점검
- [ ] HTTP 200이 나오면 주문 수집과 표준 주문행 변환 건수 확인
