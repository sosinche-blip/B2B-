# V190 운영 안내

## 1. 업체송장

1. `업체송장 선택`에서 여러 업체 송장 엑셀을 선택합니다.
2. 파일은 브라우저 앱 메모리에만 임시 보관됩니다.
3. 파일을 선택하지 않았을 때는 별도 설명박스를 표시하지 않습니다.
4. `쿠팡+토스 업로드`에서 상품준비중 주문과 매칭 미리보기를 확인합니다.
5. 최종 확인 후 채널별 송장을 등록하고 통합 결과 엑셀을 내려받습니다.

## 2. 신규 쿠폰

1. 계약서를 선택합니다.
2. API 옵션ID를 조회하고 적용할 옵션을 체크합니다.
3. `신규 쿠폰 상품명`과 `신규 쿠폰명`을 각각 입력합니다.
4. 할인방식·할인값·기간을 입력합니다.
5. 사전검증 후 신규 쿠폰 생성·적용을 실행합니다.
6. 생성된 쿠폰은 반복운영 초안으로 Supabase에 자동 저장됩니다.

## 3. 활성·대기 쿠폰 취소

1. `활성·대기 쿠폰 조회`를 누릅니다.
2. 취소할 APPLIED 또는 STANDBY 쿠폰을 체크합니다.
3. `선택 쿠폰 취소`를 누릅니다.
4. 확인창에서 쿠폰명·상태를 확인한 뒤 실행합니다.
5. 반복운영과 연결된 쿠폰은 해당 자동운영과 대기 재시도도 중지됩니다.

주의: 선택 쿠폰 취소는 실제 쿠팡 쿠폰 상태를 변경하므로 체크한 쿠폰을 반드시 확인해야 합니다.
