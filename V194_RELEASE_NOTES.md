# V194 변경사항

## 추가

- 상품준비중 상태 선택 후 쿠팡·토스 주문조회 지원
- 상품준비중 주문 체크·선택 수합 지원
- 조회 상태에 따라 선택수집 제목과 안내문구 자동 변경
- 선택목록에 실제 주문상태 표시

## 안전 분기

- 결제완료: 발주양식 생성, 채널 입력파일 생성, 정상 발주 후 상품준비중 변경
- 상품준비중: 주문목록 수합과 송장매칭 대상 반영만 수행
- 상품준비중 주문에는 발주와 상태변경을 중복 실행하지 않음

## 데이터 무결성

- 같은 주문번호·옵션ID의 상태·주소가 변경되면 최신 조회값으로 갱신
- 기존 다른 채널 및 다른 주문행을 삭제하지 않음
- V193 전체주소 보존 로직 유지
