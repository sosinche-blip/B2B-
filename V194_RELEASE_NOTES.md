# V194 변경사항

## 1. 일일 운영 점검판

간편운영 화면에서 결제완료, 현재 수집주문, 오늘 발주완료, 미매핑, 주소 차단·주의, 송장등록 준비, 미해결 실패, 상품준비중을 한 번에 확인합니다.

`마감보고서 다운로드`는 다음 시트를 포함한 엑셀을 생성합니다.

- 오늘운영요약
- 운영점검
- 주소품질
- 실패재처리

## 2. 실패 재처리 센터

다음 실패를 브라우저 작업자료에 최대 100건 보존하고 실패 단계부터 재시도합니다.

- 쿠팡·토스 결제완료 주문조회
- 주문수집 및 업체별 발주 생성
- 전체 발주파일 생성
- 송장 매칭 미리보기
- 최종 송장 업로드

쿠폰 자동운영 실패는 기존 Supabase 실패자료를 같은 화면에서 조회해 수동 재시도하거나 확인 완료로 처리합니다.

일반 실패는 기존 `operation_audit_logs`에도 오류 요약을 남깁니다. 새 테이블은 추가하지 않습니다.

## 3. 주소 품질검사

다음 항목을 주문별로 검사합니다.

- 주소 누락 또는 임시문구
- 괄호 열기·닫기 불일치
- API 원문 상세주소가 최종 주소에서 누락된 경우
- 괄호로 끝나는 주소의 상세주소 확인 필요
- 지나치게 짧은 주소
- 숫자 번지 또는 건물번호 미확인
- 우편번호 누락
- 연락처 누락

`차단` 항목은 발주 사전검증에도 연결되어 잘못된 주소로 발주파일이 생성되는 것을 막습니다. `주의` 항목은 운영자가 확인한 뒤 진행할 수 있습니다.

## 4. 유지되는 기능

- V193 주소1·전체주소·상세주소 결합
- 옵션별 독립 쿠폰 생성과 24시간 반복운영
- 활성·대기 쿠폰 선택 취소
- 선택 주문 자동 업체분류 및 ZIP 다운로드
- 업체송장 매칭 미리보기와 최종 업로드
