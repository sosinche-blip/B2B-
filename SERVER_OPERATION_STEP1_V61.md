# 서버 운영 1단계 안내 V168

Supabase를 이용해 1일 임시 작업자료, 영구 설정, 운영로그, 서버 저장소 정리 기록을 관리합니다.

## 테이블

- `operation_temp_sessions`: 주문·송장 등 당일 작업자료 임시보관
- `operation_persistent_settings`: 매핑·양식·쿠폰·스케줄 설정 영구보관
- `operation_audit_logs`: 서버 점검, 쿠폰 스케줄러, 저장소 정리, 중복방지 로그

## 자동 실행 대상

- 쿠팡 24시간 쿠폰 취소
- 쿠팡 24시간 쿠폰 신규 생성
- 서버 저장용량 점검·만료자료 정리

주문수집, 업체별 발주, 송장 업로드는 사용자가 버튼을 눌러 실행합니다.
