# B2B Operation V188

V188은 V187 쿠팡 쿠폰 자동운영을 유지하면서 다음 운영 흐름을 추가·정리한 버전입니다.

1. 앱 시작 시 쿠팡·토스 API 현황 자동조회
   - 쿠팡 결제완료
   - 토스 결제완료
   - 쿠팡 상품준비중
   - 토스 상품준비중
   - PC·모바일 모두 4개 현황을 한 줄로 표시
2. 주문 선택수집
   - `쿠팡 주문조회`, `토스 주문조회`로 결제완료 상품만 조회
   - 체크박스와 상품명/옵션명만 표시
   - 체크한 주문만 `선택 주문 수집`으로 매핑·발주 처리
3. B2B 업체 바로가기
   - 업체 URL과 로그인ID를 앱 설정에 보관
   - 모바일에서 업체 버튼을 누르면 로그인 페이지를 열고 로그인ID를 클립보드에 복사
   - 비밀번호는 Chrome·삼성 인터넷 비밀번호 관리자의 자동완성 사용
4. 쿠폰 설정 자동저장
   - 선택 쿠폰 반영, 계약·쿠폰 선택, 사전검증, 활성화, 중지, 삭제 내용을 Supabase에 자동 저장
   - 쿠폰관리의 수동 `브라우저 저장`, `서버 저장` 버튼 제거

Ncloud는 기존 V187 최소 고정 IP 게이트웨이를 그대로 사용합니다. V188 전체 소스를 Ncloud에 다시 배포하지 않습니다.
