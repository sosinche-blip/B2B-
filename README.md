# B2B Operation V184

Cloudflare R2 발주폴더와 Ncloud 고정 IP 게이트웨이를 사용하는 운영본입니다.

핵심 흐름은 다음과 같습니다.

1. `업체송장 선택` 버튼에서 여러 업체 송장 엑셀을 선택합니다.
2. 선택 파일은 Cloudflare R2 발주폴더에 저장됩니다.
3. `쿠팡+토스 업로드` 버튼을 누르면 쿠팡·토스 상품준비중 주문을 조회합니다.
4. 발주폴더 업체 송장과 주문을 자동 매칭합니다.
5. 채널별 운송장 정보를 쿠팡·토스 API에 등록합니다.
6. 업체 송장, 채널 입력파일, 확인표를 묶은 ZIP을 자동 다운로드합니다.

배포와 운영은 `DEPLOY_CLOUDFLARE_V184.md`, `OPERATIONS_GUIDE_V184.md`, `V184_RELEASE_NOTES.md`를 참고하세요.
