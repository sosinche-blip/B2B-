# B2B Operation V186

쿠팡·토스 주문수집, 옵션ID 매핑, 업체별 발주, 업체송장 임시매칭과 채널별 운송장 등록을 지원하는 운영본입니다.

## V186 핵심 변경

1. 간편운영의 쿠팡·토스 주문 수량은 결제완료 상태만 계산합니다.
2. 간편운영에서 발주업체와 송장등록 준비 수량을 제거했습니다.
3. 주문관리의 날짜별 반복 진단을 조회기간·상태별 요약으로 표시합니다.
4. 쿠팡+토스 최종 업로드 후 날짜가 포함된 전체처리 결과 엑셀 1개만 다운로드합니다.
5. 전체처리 결과 엑셀 안에 처리요약, 전체처리결과, 쿠팡업로드, 토스업로드, 미매칭확인 시트를 포함합니다.
6. 업체송장은 브라우저 메모리에만 임시 보관하며 새로고침 시 삭제됩니다.
7. Ncloud는 기존 고정 IP API 게이트웨이 역할만 유지합니다.
