# B2B Operation V191

V191은 V190의 주문·송장·쿠폰 자동운영을 유지하면서 다음 운영 흐름을 명확히 한 버전입니다.

- 결제완료 선택목록에 상품명·옵션명과 구매수량 표시
- 선택 주문 수집 시 옵션ID 매핑으로 업체별 발주양식 자동 분류
- 발주양식·채널 입력파일·검증표를 ZIP으로 자동 다운로드
- 쿠팡 신규쿠폰 입력을 옵션ID 조회표 안으로 통합
- 매칭자료의 업체상품명을 쿠폰 옵션 확인자료로 사용
- 옵션별 상품명·할인값·할인구분 입력
- 자동운영 활성화 시 신규 쿠폰 즉시 생성·적용
- 즉시 생성된 쿠폰은 다음 날부터 23:50 취소·23:51 재발행 반복
