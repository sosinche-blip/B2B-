# B2B Operation V185

업체 송장파일을 서버에 저장하지 않고 브라우저 앱 메모리에만 임시 보관하는 송장 처리 운영본입니다.

## 핵심 흐름

1. `업체송장 선택`에서 여러 업체 송장 엑셀을 선택합니다.
2. 앱이 택배사·운송장번호·주문정보를 읽어 브라우저 메모리에만 임시 보관합니다.
3. `쿠팡+토스 업로드`를 누르면 쿠팡·토스 상품준비중 주문을 조회하고 임시 송장과 매칭합니다.
4. 매칭 결과를 먼저 확인합니다. 이 단계에서는 외부 API를 실행하지 않습니다.
5. `최종 업로드`를 누르면 정상 매칭 건만 쿠팡·토스에 등록합니다.
6. 쿠팡 결과, 토스 결과, 미매칭 확인, 전체 처리결과 엑셀과 ZIP을 다운로드합니다.
7. 성공 후 임시 송장은 앱에서 삭제되며, 새로고침 시에도 자동 삭제됩니다.

업체 송장 원본은 Cloudflare R2, Supabase, Ncloud에 저장하지 않습니다. Ncloud는 쿠팡·토스 고정 IP API 호출만 담당합니다.
