# B2B Operation V192

V192는 주문수집·매핑·업체별 발주·송장등록·쿠팡 쿠폰 자동운영을 유지하면서 신규 쿠폰을 **옵션별 독립 조건**으로 등록하도록 정리한 운영본입니다.

핵심 변경:

- 상품명은 옵션별 직접 수정
- 쿠폰명 입력란 추가
- 옵션마다 쿠폰명·할인값·할인구분을 다르게 설정
- 선택한 옵션을 작성한 조건 그대로 각각 독립 쿠폰으로 즉시 생성·적용
- 다음 날부터 23:50 취소, 23:51 재발행
- 중복된 쿠폰 수동편집 UI와 과거 로컬 Windows/Ncloud 실행파일 제거
- Cloudflare 앱과 Ncloud 고정 IP 게이트웨이를 별도 패키지로 분리

배포 문서는 `DEPLOY_CLOUDFLARE_V192.md`, 운영 절차는 `OPERATIONS_GUIDE_V192.md`, 검토 결과는 `V192_REVIEW_REPORT.md`를 확인하세요.
