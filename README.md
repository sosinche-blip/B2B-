# B2B Operation V190

V190은 V189의 주문·송장·쿠폰 자동운영을 유지하면서 쿠폰관리와 간편운영 UI를 정리한 버전입니다.

1. 업체송장 설명박스 정리
   - 업체송장을 선택하지 않은 상태에서는 `업체송장 앱 임시저장` 설명박스를 표시하지 않습니다.
   - 파일을 선택하거나 매칭 미리보기가 있을 때만 간단한 파일·매칭 상태를 표시합니다.
2. 신규 쿠폰 상품명 입력
   - 신규 쿠폰 등록 화면에 `신규 쿠폰 상품명` 입력란을 추가했습니다.
   - 상품명은 사전검증 필수값이며 생성·옵션 적용·반복운영 템플릿에 동일하게 반영됩니다.
3. 활성·대기 쿠폰 선택 취소
   - `활성·대기 쿠폰 조회`로 APPLIED와 STANDBY 쿠폰을 함께 조회합니다.
   - 체크한 쿠폰만 `선택 쿠폰 취소`로 실제 취소합니다.
   - 반복운영과 연결된 쿠폰은 해당 쿠폰의 자동운영 및 대기 재시도도 함께 중지합니다.
4. 중복·위험 버튼 정리
   - 쿠폰관리 상단의 `즉시 24시간 쿠폰 생성`은 신규 쿠폰 생성·적용 및 자동운영과 중복되어 삭제했습니다.
   - 범위가 불명확한 `즉시 직전쿠폰 취소`는 삭제하고, 명시적으로 체크한 활성·대기 쿠폰만 취소하는 기능으로 교체했습니다.
