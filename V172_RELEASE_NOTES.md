# V172_MOBILE_SHIPMENT_SAFETY_LOCK 릴리스 노트

## 1. 릴리스 목적
V171의 모바일 단계 잠금판을 유지하면서, 송장 등록 단계에서 발생할 수 있는 오등록 위험을 줄이기 위해 API 업로드 전 안전검증 잠금 기능을 추가했습니다.

## 2. 핵심 변경사항

### 2.1 송장 업로드 안전검증판 추가
- 간편운영 화면에 `V172 송장 업로드 안전검증` 패널 추가
- 차단, 주의, 등록준비, 업로드잠금 상태를 모바일에서 즉시 확인
- 상위 12건의 위험 항목을 카드/표 흐름에서 바로 확인

### 2.2 API 업로드 전 차단 기준 추가
다음 항목이 있으면 쿠팡·토스 송장 API 업로드를 실행하지 않습니다.

- 쿠팡 필수ID 누락: 묶음배송번호, 주문번호, 옵션ID
- 토스 필수ID 누락: 주문상품번호
- 택배사 또는 운송장번호 누락
- 중복후보확인, 미매칭, 확인필요 매칭
- 동일 운송장번호가 서로 다른 수취인에게 배정된 경우

### 2.3 주의 기준 추가
차단은 아니지만 업로드 전 확인이 필요한 항목을 별도로 표시합니다.

- 상품명 2글자 단독 매칭
- 성명 단독 매칭
- 동일 수취인의 복수 주문에 같은 운송장번호가 사용된 경우
- 등록준비가 아닌 행은 업로드 제외로 표시

### 2.4 송장 안전검증표 다운로드
- 파일명: `B2B_송장업로드_안전검증_V172_날짜_범위.xls`
- 요약 시트: 차단, 주의, 정상 건수
- 상세검증 시트: 채널, 주문번호, 수취인, 상품명, 택배사, 운송장번호, 매칭방식, 상세 사유, 원본파일

### 2.5 업로드 버튼 잠금
- 안전검증 차단 건이 있으면 간편운영의 쿠팡+토스 업로드 버튼 비활성화
- PC 발주폴더 자동매칭 경로와 API 상품준비중 재수집 경로 모두 업로드 직전 안전검증 적용
- 차단 발생 시 API 호출 없이 안전검증표와 확인표를 먼저 생성

## 3. 유지 기능
- 쿠팡/토스 주문 수집
- 옵션ID 기준 자동 매핑
- 미매핑 카드형 수정
- 업체별 발주 엑셀 ZIP 다운로드
- 업체 송장 엑셀 업로드
- 주문번호 → 성명+주소 앞 2단어 → 성명 → 상품명 2글자 이상 송장 매칭
- 쿠팡·토스 송장등록 파일 생성
- 쿠팡 즉시할인쿠폰 23:50 취소, 23:51 적용
- 스케줄러 및 서버 저장용량 정리
