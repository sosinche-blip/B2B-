# V186 Cloudflare 배포

## 1. 배포 대상

- Cloudflare Pages: 웹 UI
- Cloudflare Worker: API 라우팅과 Supabase 연동
- Ncloud: 기존 고정 IP 쿠팡·토스 API 게이트웨이 유지

## 2. 배포

1. V186 소스를 GitHub 저장소 `main` 브랜치에 반영합니다.
2. GitHub Actions 또는 기존 배포 절차로 Pages와 Worker를 배포합니다.
3. 배포 완료 후 브라우저에서 `Ctrl + Shift + R`로 강력 새로고침합니다.

## 3. 확인 항목

1. 간편운영에 쿠팡 결제완료, 토스 결제완료, 쿠팡 상품준비중, 토스 상품준비중만 표시되는지 확인합니다.
2. 주문관리 진단이 날짜별 반복 대신 수집요약으로 표시되는지 확인합니다.
3. 업체송장 선택 시 서버·R2 저장 호출이 발생하지 않는지 확인합니다.
4. 최종 업로드 후 날짜가 포함된 전체처리 결과 엑셀 1개만 다운로드되는지 확인합니다.
5. 결과 엑셀의 5개 시트를 확인합니다.

Ncloud에는 V186 전체 압축파일을 배포하지 않습니다.
